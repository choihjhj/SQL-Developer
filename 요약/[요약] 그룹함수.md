# 그룹 함수
GROUP BY ROLLUP(),CUBE(),GROUPING SETS()    

# ROLLUP(A,B)
- 전체합계, A소계, AB소계   
  A B    
  A 널     
  A B    
  A 널        
  널널     
# CUBE(A,B)
- 전체합계, A소계, B소계, AB소계    
A B     
A 널     
널 B     
널 B   
널 B   
널널   
# GROUPING SETS(A,B) <-> GROUPING(A) 랑 다름!
- A소계, B소계     
A 널     
널 B

# GROUPING(A)
- 널이면 1,아니면 0
- 단일인자여야함
